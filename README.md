# Мероприятия в Усадьбе

Профессиональная CRM-система для управления event-мероприятиями с финансовой аналитикой и калькулятором рентабельности.

## Возможности

- **Календарь мероприятий** — вид по месяцу, неделе и дню
- **Создание и редактирование** — полная форма с деталями мероприятия
- **Калькулятор рентабельности** — расчёт ROI, маржинальности, точки безубыточности в реальном времени
- **Финансовая аналитика** — KPI карточки, графики доходов/расходов, pie chart структуры затрат
- **Поиск и фильтрация** — по названию, статусу, типу и диапазону дат
- **Экспорт** — CSV и Excel со всеми финансовыми данными
- **Тёмная тема** — полная поддержка dark mode
- **Автосохранение** — все данные хранятся в LocalStorage

## Технологии

- **React 18** + **TypeScript**
- **Vite** — сборка
- **TailwindCSS** — стили
- **Zustand** — управление состоянием
- **Recharts** — графики
- **Framer Motion** — анимации
- **Lucide React** — иконки
- **date-fns** — работа с датами
- **react-hot-toast** — уведомления
- **xlsx** — экспорт в Excel

## Установка и запуск

```bash
# Перейди в папку проекта
cd estate-events

# Установи зависимости
npm install

# Запусти dev-сервер
npm run dev
```

Приложение откроется на `http://localhost:5173`

## Сборка для продакшена

```bash
npm run build
npm run preview
```

## Формулы расчёта

| Показатель | Формула |
|---|---|
| Общий доход | Билеты × Цена + Спонсоры + Доп. доход + Бар + Мерч |
| Чистая прибыль | Доход − Расход |
| ROI | (Прибыль / Расход) × 100% |
| Маржинальность | (Прибыль / Доход) × 100% |
| Точка безубыточности | Расход / Цена билета |
| Прибыль/посетителя | Прибыль / Кол-во билетов |

## Цветовая маркировка мероприятий

- 🔵 **Запланировано** — синий
- 🟢 **Подтверждено** — зелёный
- 🟣 **Завершено** — фиолетовый
- 🔴 **Отменено** — красный

## Рентабельность

- 🟢 **Прибыльное** — ROI > 10% и прибыль > 0
- 🟡 **Зона риска** — прибыль около нуля
- 🔴 **Убыточное** — отрицательная прибыль
